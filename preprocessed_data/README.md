# Preprocessed Data

This directory contains the preprocessed datasets required for training and evaluation.

## Downloads

Download the following zip files from the shared drive and extract them into this directory:

- `physionet.zip`: Preprocessed PhysioNet dataset (images, masks, metadata).
- `mbh_seg.zip`: Preprocessed MBH segmentation dataset.
- `MBH_Seg_scan.zip`: Preprocessed MBH scan-level dataset.
- `cq500.zip`: Preprocessed CQ500 dataset.

Please download the RSNA dataset from [Kaggle](https://www.kaggle.com/c/rsna-intracranial-hemorrhage-detection) and run the preprocessing script for usage

## Directory Structure

```
preprocessed_data/
├── rsna/
├── physionet/
```
